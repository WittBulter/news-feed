跟随教程搭建基于Angular的Electron的爬虫应用

### 简介
Angular2推出有一段时间，很多人都开始尝试使用它构建一些有意思的应用，但目前国内教程非常稀少，很少有人争对代码风格、
架构、框架之间的结合、编程细节来分析。这里我结合了Electron来演示如何构建一个桌面应用。  
我希望能够丰富大家的编程经验，教程中添加了一些额外的常见场景，希望你能够跟随教程快速掌握Angular2与Electron。
